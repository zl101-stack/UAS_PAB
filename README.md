Nama Aplikasi : Dana Bijak

Nama Anggota Kelompok : Calvin Owen (03081240024), Euwin Gozalie (03081240022), Jacques Alessandro (03081240025), Princess Kosim (03081240019), Vincent Colin Robert (03081240010)

SDGs yang dipilih : No Poverty

Deskripsi singkat aplikasi : Dana Bijak adalah aplikasi pencatatan keuangan yang dirancang untuk membantu pengguna mengelola keuangan pribadi secara lebih teratur, mudah, dan efisien. Melalui aplikasi ini, pengguna dapat mencatat pemasukan, pengeluaran, serta memantau kondisi keuangan secara berkala sehingga dapat membuat keputusan finansial yang lebih bijaksana.

Fitur utama aplikasi : login/register, dashboard, pencatatan pemasukan, pencatatan pengeluaran, atur budget, target tabungan, grafik keuangan, notifikasi, profile, video edukasi

Cara menjalankan aplikasi :
1. buka aplikasi android studio kemudian jalankan code yang sudah ada, kemudian tunggu hingga emulator muncul.
2. setelah emulator muncul maka akan diarahkan ke page login aplikasi dana bijak, jika belum ada akun maka registrasi akun terlebih dahulu
3. setelah itu pengguna akan diarahkan ke halaman dashboard yang memiliki beberapa fitur utama
4. terdapat fitur pemasukan, pilih fitur pemasukan jika pengguna ingin mencatat pemasukannya.
5. terdapat fitur pengeluaran, pilih fitur pengeluaran jika pengguna ingin mencatat pengeluarannya.
6. terdapat fitur budget bulanan, pilih fiitur budget bulanan jika pengguna ingin mengontrol budget bulanan.
7. terdapat fitur target tabunngan, pilih fitur target tabungan jika pengguna ingin mengatur target tabungan.
8. terdapat fitur grafik keuangan, pilih fitur tersebut jika pengguna ingin melihat grafik keuangan.
9. terdapat fitur notifikasi, sehingga pengguna mendapatkan notifikasi.
10. terdapat fitur profile, pilih fitur profile jika pengguna ingin mengedit informasi tetang akun.
11. terdapat fitur video edukasi, pilih fitur tersebut jika pengguna ingin menambah pengetahuan tentang keuangan.
